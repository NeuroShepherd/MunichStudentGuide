# Munich Student Guide

This project aims to create a short manual for students intending to move to Munich, particularly those coming from abroad. 

## Topics Covered

- Visas
- Residence Permits
- Apartments
  - Searching
  - Anmeldung (residence registration)
- Banking
  - Sperrkonto
  - Regular Accounts
  - Transfers
- Work Regulations
- Insurances
  - Health
  - Liability (Haftplichtversicherung)
- Postal Services
  - DHL
  - Deutsche Post
- Navigating
  - Munich
  - Bayern
  - Germany
- Other Topics
  - Driver's Licenses
  - ...
